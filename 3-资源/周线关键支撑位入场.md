# 周线关键支撑位入场

## 核心概念
当个股在周线级别回踩到**多次验证过**的关键支撑位时，结合看好的方向逻辑，是可靠的买入开仓时机。

## 详细阐述
### 条件分解
- **时间级别**: 周线（周K线级别判断，周期较长，信号可靠性高）
- **支撑验证次数**: 同一支撑位被周线验证过多次（≥2次以上触碰反弹）
- **入场配合**: 配合选股逻辑（看好方向）共同决策

### 执行规则
1. 先在周线图上找到历史多次触碰反弹的水平支撑位
2. 观察价格再次接近该支撑位时，缩量/企稳可考虑入场
3. 入场需配合方向逻辑（如看好物理AI方向）

### 核心理由
- 周线级别的周期长，主力资金运作痕迹更清晰
- 多次验证的支撑位，假突破概率低
- 方向逻辑提供基本面/产业面支撑，多维度共振

## 为什么重要
将入场建立在**技术面（周线支撑）+ 基本面/产业面（看好方向）**双重依据上，减少凭感觉交易，提高胜率。

## 反例/边界
- 周线支撑被放量跌破时，应立即止损，不要死守
- 如果支撑位在周线级别已被第三次测试，力度可能减弱
- 个股流动性差时，支撑位可能不有效
