# 美联储鸽派与贵金属行情启动

## 核心概念
当美联储主席发表鸽派言论 + 经济数据（如非农）不及预期，两者共振时，黄金白银很可能启动一波大行情。

## 详细阐述
### 共振条件
1. **数据端**: 就业/通胀数据不及预期，经济放缓信号出现
2. **政策端**: 美联储主席或官员发表偏向宽松/鸽派的言论
3. **两者同时出现** → 降息预期快速升温 → 黄金白银受益

### 2026-07-01 观察
- 小非农数据不及预期 ✓
- 美联储主席发表鸽派言论 ✓
- 黄金白银暴力拉升约100美元 ✓
- 判断：黄金行情可能即将启动

### 贵金属上涨逻辑
- 降息预期 → 美元走弱 → 以美元计价的黄金上涨
- 降息 → 实际利率下降 → 持有黄金的机会成本降低
- 经济放缓 → 避险情绪升温 → 资金流入贵金属

## 为什么重要
如果能提前识别"数据不及预期+美联储鸽派"的共振信号，可以在黄金行情启动前布局，而不是在暴涨后追高。
