# 敏感词检测小程序

## 一句话
输入文案 → 检测敏感词 + 标红定位 → 给出替换建议。

## 核心功能（MVP）
**基础版（免费）**
- 文本输入（手动 / 粘贴）
- 敏感词检测 & 标红展示
- 基础词库匹配
- 敏感词数量统计

**付费版**
- 替换建议 / 一键替换
- 深度检测（更多词库维度：违禁/广告/政治敏感）
- 全文改写（AI 润色绕过敏感词）

## 用户画像
- 小红书/抖音博主
- 电商运营（商品详情/标题审核）
- 微商/私域从业者

## 竞品参考
- 零克查词
- 词爪
- 句易网
- 敏感词检测类似工具

差异化方向：做得更轻、更快、微信内直接用。

## 技术方案
- 前端：小程序原生 / Taro
- 词库：基于 [Sensitive-lexicon](https://github.com/konsheng/Sensitive-lexicon)（3.6k star, MIT, 持续更新）
- 匹配算法：Trie / DFA（轻量）或 AC自动机（大规模匹配）
- 付费：微信支付
