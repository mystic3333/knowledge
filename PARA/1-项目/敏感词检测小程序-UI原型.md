# 敏感词检测小程序 — UI原型

## 首页结构

```
┌──────────────────────────┐
│  ← 敏感词检测            │  标题栏
├──────────────────────────┤
│  [输入文案...]            │  输入区（多行文本框）
│                          │   placeholder: "粘贴或输入要检测的文案..."
├──────────────────────────┤
│  [检测]                  │  主按钮（居中，品牌色）
├──────────────────────────┤
│  检测结果: 发现 3 个敏感词 │  结果概览
├──────────────────────────┤
│  ⚠ 代购                   │  敏感词列表
│  ⚠ 微信号                 │  标红 + 上下文展示
│  ⚠ 最低价                 │  点击可查看建议
├──────────────────────────┤
│  [深度检测 →]             │  付费引导入口
├──────────────────────────┤
│  今日免费: 3/3            │  底部状态
│  [开通会员 9.9/月]        │
└──────────────────────────┘
```

## 主要页面
1. **首页** — 检测入口 + 结果展示
2. **检测详情** — 每个敏感词的位置、上下文、替换建议
3. **会员页** — 套餐展示、支付
4. **历史记录** — 最近检测记录

## 配色参考
- 主色: 橙红（警示感，#FF6B35）
- 背景: 白色 / 浅灰
- 危险: #E53935（标红敏感词）
- 安全: #43A047（全绿通过时展示）
