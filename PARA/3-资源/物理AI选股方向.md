# 物理AI选股方向

## 核心概念
**物理AI（Physical AI）** 是指将人工智能技术落地到物理世界中的方向，包括机器人、工业自动化、智能制造等。这是一个产业趋势明确、政策支持力度大的选股赛道。

## 详细阐述
### 涵盖领域
- **工业机器人**: 汇川技术、埃斯顿等
- **人形机器人**: 执行器、传感器、电机等相关产业链
- **工业自动化**: PLC、伺服系统、控制器等
- **智能制造**: 数字孪生、工业互联网

### 为什么选这个方向
- 产业趋势确定：AI从"脑子"向"身体"延伸
- 国内制造业升级的必经之路
- 政策持续加码（机器人产业政策、设备更新等）
- 汇川技术在这些领域有核心卡位

### 选股逻辑应用
- **大方向** = 物理AI
- **选个股** = 汇川技术（工业自动化+机器人龙头）
- **入场时机** = 周线关键支撑位（技术面配合）

## 为什么重要
有明确的选股方向，就不会随意买股票。物理AI是一条长赛道，可以持续挖掘标的。
